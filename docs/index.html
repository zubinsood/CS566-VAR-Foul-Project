VAR Foul Recognition

Automated Multi-View Soccer Foul Classification using Deep Learning

Read Proposal View Results

The Dataset: SoccerNet MV-Foul

Our system is trained on the SoccerNet Multi-View Foul Recognition dataset, which contains thousands of video clips from professional soccer matches captured from multiple camera angles.

Above: The same foul event captured from three different camera angles. Our multi-view model fuses information from multiple views to improve classification accuracy.

The Problem

In soccer, referees must make split-second decisions about fouls. With the introduction of Video Assistant Referee (VAR), multiple camera angles are reviewed to make fairer calls. Our goal is to automate this process using computer vision.

Classification Tasks

Foul Severity (4 classes)

  • No Offence
  • Offence + No Card
  • Offence + Yellow Card
  • Offence + Red Card

Foul Type (8+ classes)

  • Tackling
  • Standing Tackling
  • High Leg
  • Holding, Pushing, Elbowing...

Our Approach

We implemented two baseline models to classify fouls from video clips:

  1. Single-View Baseline: A ResNet18 backbone that processes 16 frames from a single camera view.
  2. Multi-View Baseline: Extends the single-view model to fuse features from 2 synchronized camera views, improving foul type classification.

The multi-view approach showed improved foul-type balanced accuracy compared to single-view, supporting the idea that multiple camera angles provide more information for classifying fouls.
